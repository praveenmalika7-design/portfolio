Cap Feeder & Elevator Machine Implementation

Company Hemas Consumer Brands (Pvt.) Ltd.
Duration November 2025 —“ May 2026
Type Machine Integration & Manufacturing Project
Category Manufacturing Automation
Status Successfully Commissioned

Participated in the installation, modification, and commissioning of an imported Cap Feeder and Elevator Machine for the cream production line. The project involved adapting imported equipment to existing manufacturing processes, developing custom cap dispensing mechanisms through additive manufacturing (3D printing), conducting multiple production trials, and achieving successful machine commissioning for commercial production.

Key Technologies: SolidWorks 3D Printing Machine Installation Manufacturing Automation Production Trials Machine Vision System Cap Dispensing Mechanism Design

Engineering Challenge

The imported machine required significant adaptation to function correctly within the existing production line layout and to work with the specific cap designs used in production. The standard cap dispensing mechanism was not compatible with the production caps, requiring custom engineering of a new dispensing component.

Project Objectives

  • Successfully install and integrate the imported cap feeder and elevator machine
  • Design a custom cap dispensing mechanism compatible with production caps
  • Manufacture the dispensing component using additive manufacturing (3D printing)
  • Conduct production trials and validate machine performance

My Responsibilities

  • Participated in installation and mechanical setup of the imported cap feeder and elevator machine
  • Designed a custom cap dispensing mechanism using SolidWorks
  • Manufactured the custom component using additive manufacturing (3D printing)
  • Identified and resolved mechanical integration issues during trials
  • Coordinated with the production team during commissioning
