Semi-Automatic Shampoo Filling Machine Development

Company Hemas Consumer Brands (Pvt.) Ltd.
Duration Nov 2025 —“ May 2026 (6 months)
Status Commissioned & In Commercial Production

Contributed to the engineering team in designing, developing, and commissioning a Semi-Automatic Shampoo Filling Machine for a high-volume FMCG production environment. The project involved complete machine design using SolidWorks, control system integration using Siemens S7-1500 PLC, HMI development using Siemens WinCC, electrical documentation using EPLAN, servo drive configuration, pneumatic system design, and CIP (Clean-in-Place) system implementation. The machine successfully entered commercial production, achieving significant production capacity improvements.

Core Technologies: SolidWorks Siemens S7-1500 PLC Siemens WinCC HMI EPLAN Servo Drive & Motor Pneumatic Systems CIP System

Engineering Challenge

The production line required an efficient, hygienic, and reliable shampoo filling solution capable of integrating with existing conveyor infrastructure. The design needed to handle precise volumetric filling of viscous fluids, implement CIP for hygiene compliance, accommodate different bottle sizes, and ensure operator safety.

Project Objectives

  • Design and develop a semi-automatic shampoo filling machine for industrial production
  • Implement precise volumetric filling using lobe pump technology
  • Integrate Siemens S7-1500 PLC for automated control and sequencing
  • Develop an intuitive HMI using Siemens WinCC for operator control
  • Design electrical control panel and document using EPLAN
  • Implement CIP (Clean-in-Place) system for hygiene compliance
  • Commission the machine and validate through production trials
  • Achieve successful commercial production launch

My Responsibilities

Performed mechanical design and 3D modelling of machine components using SolidWorks
Developed detailed engineering drawings and assembly documentation
Designed and programmed Siemens S7-1500 PLC control logic using Ladder Diagram (LAD)
Developed HMI screens using Siemens WinCC for operator interface
Designed electrical control panel layout and created comprehensive electrical schematics using EPLAN
Configured servo drive and servo motor parameters for precise positioning
Designed and integrated the pneumatic system for bottle clamping and positioning
Supervised fabrication activities carried out by outsourced contractors
Conducted machine installation, commissioning, and production trials
Coordinated with Production, QA, R&D, and Safety teams throughout the project
Managed engineering design modifications based on production trial feedback

Engineering Design Process

  1. Requirement Analysis: Collaborated with Production and QA teams to define filling accuracy, hygiene, and throughput requirements
  2. Conceptual Design: Developed machine concept incorporating lobe pump filling, conveyor integration, and servo positioning
  3. 3D Modelling & Drawings: Created detailed SolidWorks models and engineering drawings for fabrication
  4. Electrical Design: Designed control panel and electrical schematics using EPLAN
  5. PLC Programming: Programmed control logic in Ladder Diagram for filling sequences, CIP, safety interlocks
  6. HMI Development: Designed operator interface screens in Siemens WinCC
  7. Fabrication Supervision: Coordinated with contractors during machine fabrication
  8. Installation & Commissioning: Installed, wired, and commissioned the machine on the production floor
  9. Production Trials: Conducted filling accuracy tests, throughput validation, and CIP cycle testing
  10. Commercial Launch: Machine successfully entered commercial production
