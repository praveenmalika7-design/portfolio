Master Bag Packing Machine Control System Improvement

Company Hemas Consumer Brands (Pvt.) Ltd., Dankotuwa
Duration November 2025 – May 2026
Role Engineering Intern
Status Successfully Implemented
Contributed to a Kaizen-driven process improvement initiative within the sanitary napkin production line by modifying the control logic of the master bag packing machine to enable automatic packet rotation before the packaging stage. The implemented solution eliminated manual intervention, improved process consistency, reduced production inefficiency, and supported continuous improvement objectives.
Core Technologies: PLC Programming Kaizen Manufacturing Automation Industrial Engineering Mechanical Design Control Logic Modification

Engineering Challenge

The master bag packing machine required manual rotation of each packet before entering the packaging stage, creating a production bottleneck and increasing the risk of inconsistency. The challenge was to implement an automated packet rotation mechanism controlled through PLC logic modification, without major mechanical redesign and without disrupting ongoing production.

Objectives

  • Eliminate manual packet rotation through automated PLC control
  • Modify existing PLC program to control the pad rotation unit
  • Design and integrate the mechanical pad rotation unit with the existing machine
  • Validate the automated solution through production trials
  • Reduce cycle time and improve production line efficiency
  • Document the improvement as a Kaizen initiative

My Responsibilities

  • Analyzed existing PLC control logic and production line workflow
  • Designed the pad rotation unit mechanism
  • Modified the PLC control program to integrate automatic rotation sequencing
  • Coordinated mechanical installation of the pad rotation unit
  • Conducted testing and validation of the automated sequence
  • Documented the improvement process and outcomes as a formal Kaizen initiative
  • Trained production operators on the new automated system
